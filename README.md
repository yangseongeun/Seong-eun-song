# 🎵 Seong-eun Song Player
> Java Swing 기반 멀티스레딩 MP3 플레이어  
> UI 프리징 문제를 해결하며 비동기 처리 구조를 학습한 프로젝트

---

## 📌 프로젝트 개요
이 프로젝트는 **Java Swing 환경에서 발생하는 UI 프리징 문제를 해결**하기 위해 제작된 MP3 플레이어입니다.

기존에는 음악 재생 로직이 UI 메인 스레드에서 실행되면서  
👉 버튼 클릭 시 UI가 멈추는 문제가 발생했습니다.

이를 해결하기 위해 **멀티스레딩 구조와 비동기 처리 방식**을 적용하여  
👉 사용자 인터페이스와 음악 재생 로직을 완전히 분리했습니다.

---

## 🎯 개발 목적
C#의 `async/await` 기반 비동기 처리 구조를 이해하기 전에,  
Java에서 멀티스레딩을 직접 구현하며 **비동기 처리의 핵심 개념을 학습**하기 위해 제작했습니다.

---

## 🚀 주요 기능

- 🎧 MP3 파일 재생 / 일시정지 / 정지
- ⏭ 트랙 이동 기능
- 🧵 백그라운드 스레드 기반 음악 재생
- 🖥 UI 프리징 없이 부드러운 인터랙션 유지
- 🔄 재생 상태 관리 (Playing / Pause / Stop)

---

## 🛠 기술 스택

- Language: **Java**
- UI: **Swing**
- Audio: **Java Sound API**
- Build Tool: **Maven**
- Concurrency: **ExecutorService**

## ⚠️ 문제 상황 (Before)

기존 구조에서는 다음과 같은 문제가 발생했습니다.

- 음악 재생 로직이 **UI 스레드에서 실행됨**
- 버튼 클릭 시 **화면 멈춤 (UI Freeze)**
